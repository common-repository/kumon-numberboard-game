The Kumon Numberboard Game for Wordpress.

== Description ==

The Kumon Numberboard Game is a side bar widget that is fun to play and helps children develop their number sense. It can also be used for younger children to help them learn to count.

== Installation ==

1. Upload the 'kumon_numberboard' file to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the widget option, where you can enable this particular game in any sidebar.
